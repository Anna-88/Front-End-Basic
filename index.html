Git

Git — это зрелый, активно поддерживаемый проект с открытым исходным кодом, первоначально разработанный в 2005 году Линусом Торвальдсом, известным создателем ядра операционной системы Linux. Ошеломляющее количество программных проектов полагаются на Git для контроля версий, включая коммерческие проекты, а также проекты с открытым исходным кодом.
